Task20_seminar.py:
-----------------
Задача 20
Задайте список. Напишите программу, которая определит, присутствует ли в заданном списке строк некое число.

список: ["йцу", "фыв", "ячс", "цук", "йцукен"], ищем: "йцу", ответ: -1
список: ["123", "234", 123, "567"], ищем: "123", ответ: -1
